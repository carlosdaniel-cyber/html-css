Formatações via CSS

Dessa vez, vamos usar tags <span> com classes específicas que vão gerar formatações especiais. Acompanhe sempre o código-fonte da página.

Podemos colocar uma parte do texto em negrito usando CSS com a propriedade font-weight, apesar de que é possível usar <b> e <strong> para fazer isso diretamente pelo HTML.

Podemos também fazer uma parte do texto em itálico usando CSS com a propriedade font-style, apesar de que é possível usar <em> para fazer isso diretamente pelo HTML.

Também é possível transformar o texto com a propriedade text-transform, fazendo ele ficar todo em letras maiúsculas, ou então EM MINÚSCULAS, ou até capitalizadas independente do conteúdo usando CSS. Você só vai entender isso direito se olhar o que fiz no código-fonte.

E por fim, usando a propriedade text-decoration podemos criar uma decoração aos textos, fazendo-os ficar com uma linha abaixo, ou com uma linha acima das letras, ou até mesmo através delas, conforme a nessecidade.
